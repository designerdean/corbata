Corbata

Corbata is a base layer of rules and conventions that act as a starting point for HTML, CSS, and JavaScript for your own projects. I got the idea from Simon Collison's reusable package concept. He defines a reusable package as being:

a bumper compendium of cascading and connected CSS files, naming conventions, modules, plugins and library scripts that ensure any project led or worked on by any member(s) of a team will stay on convention, and be simpler for anyone else to step into and work with at any time.

My reusable package includes all the things I use in nearly every website I build:

Markup & Images

  • Paul Irish's conditional markup
  • Viewport Metatag to control layout in mobile browsers
  • Markup prompting <IE8 users to upgrade browsers or install Google Chrome Frame
  • Pre-sized favicon and Apple touch icons

CSS / Sass / Compass (requires an installation of the prereleases of Sass and Compass)

  • Andy Clarke's print.css
  • Google web fonts CSS tag (customizable)
  • Trevor Davis' Nested Responsive Grid (Sass)
  • Nested Media Queries (Sass)
  • Compass imports for Reset, CSS3, Typography, Utilities, and Layout
  • An attempt to follow Nicolas Gallagher's idiomatic CSS guidelines for stylesheet organization
  • Paul Irish's idea to apply box-sizing: border-box to all elements

JavaScript

  • Modernizr
  • jQuery (via the Google Code protocol-less reference)
